## Used Technologies

* Back-end: Spring (Boot, Data, Security), JPA / Hibernate, MySQL, JUnit, Developer Tools
* Front-end: HTML, React.js, Redux, Bootstrap, CSS
* Security: JWT, Github
* REST API
* Database: MySQL
* Server Build: Maven
* Client Build: npm

## Features

* Browse books without registering
* Add to cart
* Checkout Page
* Sign in
* Sign up
